👋 課程核心理念

本課程結合「地球物理知識」與「資料處理技能」。

透過 18 週的訓練，您將掌握從野外折射探勘、重力測量，到使用 Python (PyGMT, ObsPy) 進行地震監測與 AI 數據分析的完整流程。

課程三大支柱
  • 地球物理：折射、重力、板塊構造、地震學。
  • 程式技能：Python 環境建置、PyGMT 繪圖、ObsPy 資料處理。
  • 前沿應用：GitHub 協作、Hugging Face 部署、Gemini AI 應用。
